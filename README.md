# 简介
这是一个使你放置自定义树苗的模组。
#自定义模式
### COMMAND MODE
##### 介绍：该模组默认关闭，当在配置文件中启用时，可以使用 /cs-set <x> <y> <z> <树干方块> <树叶方块> <树的高度> <树叶高度> <树叶的半径> 来对指定位置的自定义树苗自定义。 `cs:custom_sapling`
##### 其他自定义方式 ： 按住潜行键右键自定义树苗方块可以自动弹出指令。(不稳定，慎用！) 
### DEFAULT MODE
##### 介绍：默认自定义模式。
##### 自定义方式：在树苗的下方一个放置的方块对应树干，下方第二个放置的方块对应树叶。
